# dotfiles

Personal configuration files for Ubuntu 24.04.

## Setup

```bash
git clone git@github.com:stevengnb/dotfiles.git ~/Documents/projects/dotfiles
cd ~/Documents/projects/dotfiles
./install.sh
```

## What's included

| Config | Description |
|---|---|
| zsh | zshrc + Powerlevel10k prompt |
| kitty | Terminal with Rose Pine theme + Tokyo wallpaper |
| tmux | Prefix `Ctrl+Space`, vi copy mode |
| nvim | LazyVim setup |
| pop-shell | Tiling config |
| ulauncher | Launcher settings + shortcuts |
| copyq | Clipboard manager config |
| fastfetch | System info display |
| dconf | GNOME keybindings + workspace shortcuts |
