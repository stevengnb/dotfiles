Prefix: Ctrl+Space

| Key                    | Action                              |
|------------------------|-------------------------------------|
| Ctrl+Space, s          | Switch between sessions             |
| Ctrl+Space, $          | Rename current session              |
| Ctrl+Space, d          | Detach session (keeps running)      |
| Ctrl+Space, c          | New window                          |
| Ctrl+Space, 0-9        | Switch to window by number          |
| Ctrl+Space, o          | Next window (repeatable)            |
| Ctrl+Space, i          | Previous window (repeatable)        |
| Ctrl+Space, ,          | Rename window                       |
| Ctrl+Space, x          | Close current pane/window           |
| Ctrl+Space, \|         | Split vertical (opens in current path) |
| Ctrl+Space, -          | Split horizontal (opens in current path) |
| Alt+Arrow              | Move between panes (no prefix)      |
| Ctrl+Space, m          | Zoom pane fullscreen (toggle)       |
| Ctrl+Space, [          | Enter copy mode (vi keys)           |
| v                      | Start selection (in copy mode)      |
| y                      | Yank/copy selection (in copy mode)  |
| Ctrl+Space, Ctrl+s     | Save session (resurrect)            |
| Ctrl+Space, Ctrl+r     | Restore session (resurrect)         |
| Ctrl+Space, r          | Reload tmux config                  |
| tmux new -s name       | New named session (outside tmux)    |
| tmux attach -t name    | Attach to session (outside tmux)    |
| tmux ls                | List all sessions (outside tmux)    |
| exit                   | Close current pane/window           |
